# Airbnb Global Performance Analysis

## 📘 Overview
This project analyzes Airbnb's global marketplace performance across 10 major cities using Power
BI. The objective was to identify business trends, customer satisfaction patterns, pricing behavior,
host performance, and growth opportunities through interactive visual analytics.

---

## 📁 Dataset
Almost 2 Lakh Rows And 20+ Columns 

---


---

## ⚙️ Project Workflow
1. **Data Loading:** Imported DataSet In Power BI  
2. **Data Cleaning:** Using Power Query Editor
3. **Data Modelling** Build Relationship Between Various Tables
4. **Dashboard Creation:** Built interactive multi-page dashboards in **Power BI**  
5. **Reporting:** Compiled a detailed report summarizing findings, trends, and recommendations.
